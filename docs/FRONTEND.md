# Frontend

## Current State

Aurora Translator is currently a CLI and library project. There is no production
web frontend in this repository.

## If A Frontend Is Added

- Build the actual workflow first, not a marketing landing page.
- Optimize for PCB translation operators: dense status, clear diagnostics,
  reproducible commands, and inspectable artifacts.
- Expose parser summaries, schema versions, diagnostics, and output paths as
  first-class UI state.
- Keep visual design restrained and work-focused.
- Verify UI changes with screenshots or browser automation before handoff.
